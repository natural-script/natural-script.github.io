Jste Features

Translated

Jste Framework syntax has been translated to various languages from all around the world to make programming easier and simpler for everyone who hasn't any programming experience.

Fast & Thrifty

Jste-Powered apps and sites only stored in one small easy-to-read HTML file and the framework itself is stored locally in a small minified HTML file which saves data and loading time, in addition to that no media will be downloaded without your permission.

Sexy

All Jste components have been deisgned carefully according to the latest modern Google Material Design guidlines thanks to Materialize and Polymer Framework.

Open Source

Because we believe that open source is the main key of the projects success, we have licensed Jste under AGPL v3.0 to give all the developers all around the world the ability to extend this project and its components and port it to their own local languages.

Fully Plain Text Syntax

Jste syntax is fully plain, simplified and translated which means that if you want to develop anything using it, all what you have to do is opening a HTML file and writing plain text in your local language which everyone familiar with your langauage can understand it without even if he/she doesn't know WHAT THE COMPUTER IS !

This will help the noob people espicially in the poor countries to take part in the fastly growing technology world and the world wide web.....
